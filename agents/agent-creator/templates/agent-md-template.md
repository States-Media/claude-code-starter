## Usage Instructions

### How to Use This Template

1. **Replace all placeholders** marked with [brackets]
2. **Remove optional sections** if not applicable (e.g., Sector Expertise for general agents)
3. **Expand sections** based on research depth
4. **Maintain formatting** for consistency
5. **Ensure minimum length** of 200+ lines

### Key Placeholders to Replace

- `[agent-name-kebab-case]`: e.g., "product-manager", "data-scientist"
- `[Agent Type]`: e.g., "Product Manager", "Data Scientist"
- `[X]+ years`: Specify experience level (10+, 15+, 20+)
- `[Sector/Industry]`: Only if sector-specific

### Section Guidelines

**Core Identity**: Establish credibility and expertise level
**Responsibilities**: 5-7 major functions, detailed
**Competencies**: Mix of technical, strategic, and soft skills
**Deliverables**: 8-12 specific outputs the agent can create
**Sector Expertise**: Include only if sector-specific
**Operating Principles**: 5-10 guiding principles
**Communication Style**: Define tone and approach
**Frameworks**: Specific methodologies and tools
**Reference Materials**: Optional but adds credibility
**How to Engage**: Step-by-step interaction process

### Quality Checklist

Before finalizing:
- ☐ All placeholders replaced
- ☐ No generic descriptions
- ☐ Specific tools and frameworks named
- ☐ Deliverables are actionable
- ☐ Communication style defined
- ☐ Engagement process clear
- ☐ 200+ lines minimum
- ☐ Professional tone throughout
