# Create Domain Squad — Domain Specialist Pipeline

> Create a squad of domain specialists without cloning a specific person.

## Execution

1. Read the clonning-chief agent at `~/.claude/squads/squad-clonning/agents/clonning-chief.md`
2. Read the workflow at `~/.claude/squads/squad-clonning/workflows/wf-domain-squad.yaml`
3. Execute: Pre-flight → Extract Domain → Build Squad → Validate
4. Same quality bar as mind clone (85/100)

## Usage

```
/squad-clonning:tasks:create-domain-squad "Paid Traffic Management"
/squad-clonning:tasks:create-domain-squad "QA Engineering"
/squad-clonning:tasks:create-domain-squad "DevOps & Infrastructure"
```

## Key Difference from Mind Clone

- No Voice DNA → replaced by Domain Voice Guide (tone/vocabulary for the domain)
- No Thinking DNA → replaced by Domain Decision Architecture (heuristics, frameworks)
- Content comes from domain literature, not a person's published material
- **Quality bar is IDENTICAL** — same 85/100 threshold, same mandatory sections
