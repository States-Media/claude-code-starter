## A

**Ad Account** — Conta de anúncios em plataformas como Meta (Facebook/Instagram), Google Ads, TikTok.

## B

**BU (Business Unit)** — Unidade de negócios independente dentro da States Media.

## C

**CLAUDE.md** — Arquivo de configuração que ensina o Claude Code como se comportar num projeto ou contexto específico.

**Claude Code** — Ferramenta de IA da Anthropic que roda no terminal. É a principal ferramenta de IA da States Media.

## L

**Landing Page (LP)** — Página de destino onde o tráfego de anúncios chega.

## M

**MCP (Model Context Protocol)** — Protocolo que permite ao Claude Code se conectar a serviços externos (Google Drive, Gmail, etc.).

## S

**Skill** — Capacidade adicional instalada no Claude Code. Funciona como um "plugin" que ensina o Claude a fazer algo específico.

## V

**VSL (Video Sales Letter)** — Vídeo de vendas usado em funnels de marketing direto.

## W

**Wiki** — Base de conhecimento da empresa, mantida pelo Claude Code usando o padrão LLM Wiki.
